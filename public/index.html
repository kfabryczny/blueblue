BlueBlue BLE beacon scanner (for Radioland beacons).

Start scanning

To start scanning call this URL (replace 0.0.0.0 with your server IP):

http://0.0.0.0:{{ . }}/start

Stop scanning

To stop scanning call this URL:

http://0.0.0.0:{{ . }}/stop

Note: you need to stop scanning before you can stop the server.

Get JSON of all beacons found

To get a JSON of all beacons detected past 90 seconds call this URL:

http://0.0.0.0:{{ . }}/beacons?last=90 Default is 60 seconds.
